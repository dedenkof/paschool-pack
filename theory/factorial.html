Факториал

Факториал числа n (n!) - произведение всех натуральных чисел, от 1 до n включительно
    n! = 1 * 2 * 3 * ... * n
    0! = 1
    В комбинаторике факториал натурального числа n
    интерпретируется как количество перестановок множеств из n элементов
    Н-р: для множества A,В,С,D иp 4-х элементов существует: 4! = 1 * 2 * 3 * 4 = 24 перестановки
    ABCD  BACD  CABD  DABC
    ABDC  BADC  CADB  DACB
    ACBD  BCAD  CBAD  DBAC
    ACDB  BCDA  CBDA  DBCA
    ADBC  BDAC  CDAB  DCAB
    ADCB  BDCA  CDBA  DCBA

  • Декремент от нашего числа до 0

    Variant 1

                    var counter = prompt("Введите число"); // Наше изначальное число
                var factorial = 1;

                document.write("Факториал  числа: " + counter + "! = ");
                // Проверка на минусовое значение, строку - введи заново
                do{
                    // Факториал 0 это 1
                    if(counter == 0){
                        factorial = 1;
                        break;
                    }

                    factorial *= counter;
                    counter--;
                    // Пока counter больше нуля выполняем итерацию
                }while (counter > 0);

                document.write(factorial);

  • Инкремент от 0 до нашего числа факториала

    Variant 2

                    var counter = prompt("Введите число"); // Наше изначальное число
                var factorial = 1;
                var i = 0;
                document.write("Факториал  числа: " + counter + "! = ");
                // Проверка на минусовое значение, строку - введи заново
                do{
                    i++;
                    // Факториал 0 это 1
                    if(counter == 0){
                        factorial = 1;
                        break;
                    }

                    factorial *= i;

                    // Пока counter больше нуля выполняем итерацию


                }while (i <= counter);

                document.write(factorial);
